# Учебная Платформа

Этот проект представляет собой учебную платформу, разработанную с использованием фреймворка Django и Django REST framework. Платформа позволяет пользователям создавать и управлять продуктами, уроками и отслеживать прогресс обучения.

## API

Проект предоставляет следующий набор API:

1. Список уроков по всем продуктам, к которым пользователь имеет доступ, с информацией о статусе и времени просмотра.

   `GET /api/products/`

2. Список уроков по конкретному продукту, к которому пользователь имеет доступ, с информацией о статусе, времени просмотра и дате последнего просмотра.

   `GET /api/products/{product_id}/`

3. Статистика по продуктам, включая количество просмотренных уроков, общее время просмотра и количество пользователей, занимающихся на продукте, и процент приобретения продукта.

   `GET /api/products/get_statistics/`

Аутентификация реализована через djoser.

### Регистрация нового пользователя

Для регистрации нового пользователя, отправьте POST-запрос на эндпоинт `/auth/users/` с данными в JSON-формате:
```
{
  "username": "example_user",
  "password": "example_password"
}
```
Для получения токена отправьте POST-запрос на эндпоинт `/token/login/` с теми же данными, вам вернется:
```
{
"refresh": "your_refresh_token_here",
"access": "your_access_token_here"
}
```
Далее используйте access токен для доступа к другим эндпойнтам.
